# ai-research-assistant

## Setup

1. Create a local `.env` file in the project root.
2. Copy the values from `env_template.txt` into `.env` and replace the placeholders with your real API keys.
3. Install dependencies with `pip install -r requirements.txt`.
4. Run the app with `streamlit run app.py`.

Do not commit your local `.env` file. The repository should only contain placeholder values in `env_template.txt`.
